# ATM

Демо-проект с реализацией банкомата, разработанный на библиотеке ОСень/Autumn в рамках [стрима на канале Веселый1С](https://www.youtube.com/watch?v=eIhWSJnRXlU)

Отличия от версии на стриме:

* [PR #1](https://github.com/nixel2007/autumn-atm/pull/1) - вместо сохранения пользователя через МенеджерСущности используется метод "активной записи", автоматически добавляемый в класс "Пользователь" при создании его через метод `МенеджерСущности.СоздатьЭлемент()` библиотеки entity (с помощью decorator).
* [PR #1](https://github.com/nixel2007/autumn-atm/pull/1) - Идентификатор пользователя изменен с автогенерируемого числового значения на собственно имя пользователя. Упрощен поиск пользователя методом `МенеджерСущности.ПолучитьОдно()`.
